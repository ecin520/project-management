### 简介

项目管理系统，使用现阶段主流技术SpringBoot、MyBatis、MySql、Redis等，采用Docker容器化部署。

| 技术            | 说明               |
| --------------- | ------------------ |
| SpringBoot      | 容器+MVC框架       |
| Spring Security | 权限框架           |
| MyBatis         | ORM框架            |
| Swagger-UI      | 文档生成器         |
| PageHelper      | 物理分页插件       |
| Redis           | 分布式缓存         |
| Docker          | 容器化部署         |
| Druid           | 数据库连接池       |
| JWT             | JWT登陆支持        |
| Lombok          | 强大的简化代码插件 |
